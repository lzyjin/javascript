태그에 접근하는 방법

자바스크립트에서 태그는 하나의 객체다 (DOM)
객체에는 태그를 표현하는 properties(속성)가 있다.
태그에 접근하는 방법은 크게 4가지. 속성값을 이용해서 태그를 가져온다.

  1. id값을 기준으로 가져오기
  2. class값을 기준으로 가져오기
  3. tag명을 기준으로 가져오기
  4. name값을 기준을 가져오기

id값을 기준으로 가져오기

태그의 id속성에 있는 값을 기준으로 태그를 가져온다
document.getElementById("id속성 값");
id 속성의 값이 일치하는 태그 한개를 가져온다.

class속성을 기준으로 값을 가져오기

태그에 설정된 class속성의 값을 기준으로 태그 객체를 가져온다.
중복값을 허용해서 같은 class명이 여러개일 경우 여러개의 객체를 불러온다.
document.getElementsByClassName("class속성값");

선택자를 이용해서 태그를 가져오기

css에서 사용한 선택자를 이용해서 태그를 지정하여 가져오는 것
#test : id가 test인
.test : class가 test인
[type=test] : type속성이 test인
div>p : div의 자식인 p
document.querySelector("선택자");
한 개의 태그만 가져오는 것 ( 다수일 경우 첫번째 태그만 ) document.querySelectorAll("선택자");
다수의 태그를 가져오는 것
