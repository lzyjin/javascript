자바스크립트 이용하기

자바스크립트로 페이지를 interactive하게 구성하려면 자바스크립트 구문을 태그 내 속성(inline), script태그 내부, 외부 js파일에 작성한다.

  1. inline 방식 : 태그 내부의 이벤트속성(on***)에 직접 javascript코드를 작성하는 것 ( 간단한 구문에만 사용 )
  2. internal 방식 : 페이지의 script태그 안에 javascript코드를 작성하는 것
  3. external 방식 : ***.js 파일에 javascript코드를 작성하고, 페이지에서 script태그를 이용해서 불러와(script src="주소") 사용 / 서버 내부에 저장된 파일이나 외부(http...주소)에서 지원하는 파일 두개 다 가능
