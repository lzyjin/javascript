데이터 출력하기

스크립트에서 데이터를 출력하는 방식 2가지
1. 페이지에 출력 ( 문구 )
2. 브라우저가 출력 ( 알림창, 개발자도구 console창에 )

  • 페이지에 출력
    1. document.writer("출력구문 || 태그 + 출력구문")
    2. 태그선택.innerHTML="출력구문"
  • 브라우저에서 출력
    1. console창 출력 : console.log("출력문구")

페이지에 출력하기

document.write()함수(메소드) 이용하기

innerHTML 속성 이용하기

태그마다 innerHTML속성을 가지고 있다. innerHTML속성은 현재 출력하고 있는 문구를 저장하는 속성이다. innerHTML속성에 저장되어 있는 문구가 화면에 출력된다. 태그.innerHTML="문구";

브라우저에 출력하기

알림창으로 출력하기 : alert()함수를 이용

console창에 출력하기 : console.log()함수를 이용
